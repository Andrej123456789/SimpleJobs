# SimpleJobs
Simple Minecraft plugin which implements jobs!

Version: 1.19.4+

## How to use
- *COMING SOON*

### Permissions
- `simplejobs.jobs` - for accessing jobs (players)
- `simplejobs.admin` - for controlling jobs (administrators)
